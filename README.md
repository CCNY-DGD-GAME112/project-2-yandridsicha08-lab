# Project 2

This project should be a 3D game and meet the requirements below. Beyond that it can be whatever game you can dream of and can build in 1 month.

**DUE 4/15 11:59 PM**

## Requirements

1. Must be in 3D!
2. Needs to use this repo created by github classroom.
3. 75% Technical requirements
    1. At least one “AI” character with a coroutine
    2. 2 or more classes that share a base class using inheritance
    3. At least one object should use raycasts to do something (gun, laser security, enemy vision)
4. 10% Choose 2 of:
    1. Singleton/GameManager
    2. Timer
    3. Score
5. 15% Game design
